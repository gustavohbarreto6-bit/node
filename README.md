Sistema de Atendimento - Feira de Tecnologia
📖 Descrição

Este projeto simula um sistema de atendimento para uma feira de tecnologia utilizando o Node.js no modo REPL.
O objetivo é praticar comandos básicos do JavaScript no terminal, incluindo variáveis, estruturas condicionais, loops, funções e manipulação de arquivos.

🚀 Funcionalidades
Exibição de mensagens no console
Registro de data e hora do atendimento
Cadastro de dados do visitante:
Nome
Idade
Quantidade de ingressos
Valor do ingresso
Validação de idade
Alerta para quantidade elevada de ingressos
Cálculo do valor total da compra
Exibição dos dados em formato de tabela
Geração de senhas de atendimento
Listagem de ingressos
Exibição de setores da feira
Função de saudação personalizada
Criação de arquivo com resumo do atendimento
🧑‍💻 Tecnologias Utilizadas
Node.js
JavaScript
Módulo nativo fs (File System)
📋 Dados Utilizados no Exemplo
Nome: Lucas
Idade: 17 anos
Ingressos: 3
Valor do ingresso: R$ 20
📊 Saída Esperada

O sistema exibe no console:

Informações do atendimento
Tabela com os dados do visitante
Senhas de atendimento
Lista de ingressos
Setores disponíveis
Mensagem de boas-vindas

Além disso, é gerado um arquivo:

📄 saida.txt

Contendo:

Nome: Lucas
Idade: 17
Ingressos: 3
Total: R$ 60
⚙️ Como Executar
Abra o terminal
Inicie o Node.js:
node
Execute os comandos manualmente no REPL
📌 Conceitos Praticados
console.log, console.table, console.warn, console.error
Estruturas condicionais (if/else)
Laços de repetição (while, for, forEach)
Funções
Arrays
Template strings
Manipulação de arquivos com fs.writeFileSync
🎯 Objetivo Educacional

Este exercício tem como finalidade reforçar o aprendizado de lógica de programação e comandos básicos do JavaScript no ambiente Node.js.
